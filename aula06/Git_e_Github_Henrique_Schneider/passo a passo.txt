primeiro: criar uma nova pasta (nome qualquer)
$mkdir (nome)

segundo: abrir a pasta e iniciar o git
abrir a pasta e clicar com o botao direito do mouse, escolher a opção git here

terceiro: no git, dentro da nova pasta vamos clonar o repositorio da Turma1-2022
$git clone (URL)
copiar a url no https://github.com/huberpoa/turma1-2022 clicando no botão verde "Code", copiar o link para ser inserido no comando $glit clone
ex. $glit clone https://github.com/huberpoa/turma1-2022

quarto: o git vai copiar o repositorio Turma1-2022 para a pasta que voce criou anteriormente
agora abra a pasta Turma1-2022
$cd Turma1-2022
note que o diretorio ja aparece como main

quinto: dentro da pasta Turma1-2022, coloque os arquivos que voce quer suber para o github. 

executando os comandos.
primeiro 
$git status
para ver o que esta acontecendo
$git add . 
para adicionar seus arquivos
$git commit -m "seu commit"
tende dar um git push
$git push oringin main
se der certo os arquios serao adicionados
